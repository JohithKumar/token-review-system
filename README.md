Token Review System

• A simple web application built using Python and Django
• Used to manage tokens such as complaints or service requests
• Users can create tokens and administrators can review and manage them
• Beginner friendly project for learning Django full stack development

Features

• User registration and login system
• Create token or request
• Admin dashboard for management
• Token status tracking
• Simple and clean user interface
• Secure authentication

Technology Used

• Frontend: HTML, CSS, Bootstrap
• Backend: Python and Django
• Database: SQLite
• Version Control: Git and GitHub

Installation Steps

• Clone the repository using git clone and open the project folder
• Create a virtual environment using python -m venv env
• Activate the virtual environment
• Install requirements using pip install -r requirements.txt
• Run migrations using python manage.py makemigrations and migrate
• Create superuser using python manage.py createsuperuser
• Run the server using python manage.py runserver
• Open the project in browser at http://127.0.0.1:8000/

Project Purpose

• To learn Django models
• To understand authentication system
• To practice CRUD operations
• To learn admin panel usage
• To understand full stack workflow
